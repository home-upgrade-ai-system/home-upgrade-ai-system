# SJ Decoration

SJ Decoration provides residential home improvement solutions throughout Greater Los Angeles, with a focus on:

- Blinds & shades
- Custom closets
- Windows & doors

## What We Offer

We help homeowners:

- understand their space and project needs
- review suitable product options
- compare styles, materials, and practical choices
- request on-site measurement and consultation
- confirm installation details and next steps
